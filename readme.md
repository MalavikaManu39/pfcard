# 📇 Profile Card UI

A clean and responsive **Profile Card UI** built using **HTML and CSS**. This project displays personal details, contact information, and social links in a visually appealing card layout.

---

## ✨ Features

* Modern card design with smooth layout
* Profile header with avatar, name, and role
* Stats section (Projects, Followers, Following)
* About section with short bio
* Personal information grid
* Contact details (Phone & Email)
* Social media buttons (LinkedIn & GitHub)
* Action buttons (Message & Connect)

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox & Grid)

---

## 📂 Project Structure

```
profile-card/
│── index.html
```

---

## 🚀 How to Run

1. Download or clone this repository
2. Open the `index.html` file in any web browser
3. The profile card will be displayed

---

## 🎨 Customization

You can easily customize:

* Name, role, and avatar initials
* Stats values
* Bio and personal information
* Contact details
* Social media links
* Colors and styling in the CSS section
